# FPS Zombie Game - Unity

First-person shooter zombie game developed in Unity with a focus on modular and scalable architecture.

##  Features
- Player controller (FPS)
- Weapon system
- Zombie AI (NavMesh)
- Wave system (Horde system)
- Mission system (extraction, objectives)
- NPC interaction system

##  Technologies
- Unity (C#)
- NavMesh AI
- Game systems architecture

##  Description
This project focuses on building scalable game systems, enemy AI behavior, and modular architecture for future expansion.
